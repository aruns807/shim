Shim

A vim inspired text editor

Download this project as a .zip file Download this project as a tar.gz file

What is shim?

It's a text editor that:

  • Is modal like vim
  • Is highly configurable like Vim
  • Understands a subset of commonly used vim motions (dw, m[buffer name], etc).
  • Has a plugin system that behaves like "real" code (none of that API/scripting stuff)

Why?

  • Contributing to open source is hard and I want to make something that is hopefully less intimidating for people to contribute to.
  • I highly doubt most people use the full feature set of Vim and creating something that is easier learn / extend could be extremely useful.

Doing so will give people a project that they can both use day to day as well as contribute to (something that I wish I had when I first started with open source).
